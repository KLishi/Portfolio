Project TV Repositorium

TV guide app

TV Repositorium is a TV guide app. I worked in team of four to build this app. We build this app utilizing TVMaze API on Angular frame work. The primary function of this app is to get details of TV shows using search function. Angular material cards are used in UI. App also contains responsive flex layout design to make it small screen friendly.

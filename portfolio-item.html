Project Connexion

Photo Sharing App

Connexion is a photo sharing social media web app. I worked in a team of four to build this app. We built all aspects of this application, from databases to the APIs, front end, styling etc.

I developed components like dashboard to register user profiles. I coded few models,routes/API for the app. We used MERN stack technology to build this app. Data is passed between components utilizing Redux store.

Features include responsive design, user profiles, media upload and media import, JWTToken authentication, password encryption, image hosting via Cloudinary, like/comment/follow functionality, and more.
